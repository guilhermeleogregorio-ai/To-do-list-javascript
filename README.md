# ✅ To-Do List

Aplicação web para gerenciamento de tarefas, desenvolvida com JavaScript, com foco na manipulação do DOM e interatividade.

---

## 🚀 Funcionalidades
- Adicionar tarefas  
- Remover tarefas  
- Marcar tarefas como concluídas  
- Atualização dinâmica da interface  

---

## 🛠 Tecnologias Utilizadas
- JavaScript  
- HTML  
- CSS  

---

## 🎯 Objetivo do Projeto
Praticar lógica de programação e manipulação do DOM, desenvolvendo uma aplicação interativa e funcional.

---

## ▶️ Como Executar

1. Baixe ou clone o repositório:

git clone https://github.com/SEU-USUARIO/todo-list-javascript

2. Abra o arquivo index.html no navegador
